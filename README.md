# AreaIQ by Hart

A neighborhood research tool for Indiana homebuyers, helping prospective clients evaluate areas they're considering before they buy.

**Live site:** https://klhart88.github.io/areaiq/

**Created by:** Kelvin Hart, Fathom Realty (Noblesville, IN)

## Features (v1)

- Commute estimates (peak vs. off-peak)
- School district + neighborhood demographics
- Area-level property tax estimate
- Utility & service providers
- Nearby restaurants, parks, and amenities
- Future development trends
- Side-by-side comparison of up to 3 areas
